3. Javalin.
4. Otros.



Para instalar Kotlin se puede usar SDKMAN!

```java
$ sdk install kotlin
```

Mostrar versión instalada:

```java
$ kotlinc -version
```

El proyecto usa **Gradle** para construir y ejecutar el proyecto.
